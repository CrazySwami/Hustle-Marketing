# Work Tasks Command

Execute tasks from a task list one at a time.

## Instructions

1. Ask the user which task list to work from (check `tasks/tasks-*.md` files)
2. Find the first incomplete task (has `- [ ]`)
3. Implement that ONE task
4. Run any required tests/checks
5. Mark the task complete by changing `- [ ]` to `- [x]`
6. STOP and ask user to verify before continuing

## Important

- ONE task at a time
- Mark complete IMMEDIATELY after finishing
- Browser-verify any UI changes
- Never skip verification steps
