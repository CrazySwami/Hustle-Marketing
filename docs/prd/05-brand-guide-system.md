# Brand Guide System PRD

## Document Info
- **Feature:** AI-Powered Brand Guide System
- **Priority:** P0 (Core Feature)
- **Version:** 1.0
- **Status:** Partially Implemented
- **Current State:** Basic brand guide generator exists

---

## 1. Overview

### 1.1 Feature Summary
A comprehensive brand identity system that enables users to create, manage, and automatically apply brand guidelines across all content. The AI extracts brand elements from websites, logos, and existing assets, then intelligently applies them to all designs, templates, and AI-generated content.

### 1.2 Problem Statement
Maintaining brand consistency across social media is challenging. Marketers manually apply colors, fonts, and styles to every piece of contentâ€”often getting it wrong. Agencies juggle multiple client brands, leading to mix-ups. Current tools require manual brand setup that most users don't complete, resulting in off-brand content.

### 1.3 Solution
An AI-first brand system that:
1. **Extracts** brand identity automatically from website/assets
2. **Applies** brand elements intelligently to all designs
3. **Enforces** consistency across team members and content types
4. **Evolves** by learning from user corrections and preferences

### 1.4 Success Metrics
| Metric | Target |
|--------|--------|
| Brand guide completion rate | 85% |
| Brand-applied content rate | 95% of exports |
| Brand consistency score | 4.5/5 user rating |
| Time to create brand guide | < 3 minutes |

## 2. User Stories

### 2.1 Primary User Stories

#### US-1: Create Brand from Website
> **As a** small business owner
> **I want** the AI to extract my brand from my website
> **So that** I don't have to manually specify every brand element

**Acceptance Criteria:**
- [ ] User enters website URL
- [ ] AI extracts colors, fonts, logo, and style
- [ ] Preview shows extracted brand guide
- [ ] User can adjust any extracted element
- [ ] Process completes in < 30 seconds

#### US-2: Create Brand from Logo
> **As a** startup founder
> **I want** to generate brand colors from my logo
> **So that** I get a cohesive palette even without a website

**Acceptance Criteria:**
- [ ] User uploads logo file
- [ ] AI extracts dominant colors
- [ ] AI suggests complementary colors
- [ ] Logo is stored for auto-placement
- [ ] Multiple logo variants supported (light/dark)

#### US-3: Auto-Apply Brand to Design
> **As a** marketing manager
> **I want** my brand colors applied to every new design
> **So that** I never accidentally use wrong colors

**Acceptance Criteria:**
- [ ] New projects use active brand by default
- [ ] Templates adapt to brand colors
- [ ] AI-generated content uses brand voice
- [ ] User can override for specific elements

#### US-4: Manage Multiple Brands
> **As an** agency account manager
> **I want** to switch between client brands easily
> **So that** I never mix up client brand elements

**Acceptance Criteria:**
- [ ] Create unlimited brand guides
- [ ] Quick-switch brand selector
- [ ] Visual brand thumbnails
- [ ] Last-used brand remembered per project

#### US-5: Brand Voice for AI
> **As a** brand manager
> **I want** AI-generated copy to match our brand voice
> **So that** content sounds consistent whether human or AI wrote it

**Acceptance Criteria:**
- [ ] Define brand voice attributes
- [ ] AI uses voice in all text generation
- [ ] Example phrases for reference
- [ ] Tone variations (formal/casual) within voice

### 2.2 Secondary User Stories

#### US-6: Brand Compliance Check
> **As a** creative director
> **I want** the system to warn when content is off-brand
> **So that** we catch mistakes before publishing

#### US-7: Import Brand from Figma
> **As a** designer
> **I want** to import my Figma brand styles
> **So that** I don't recreate what already exists

#### US-8: Brand Usage Analytics
> **As a** brand owner
> **I want** to see which brand elements are used most
> **So that** I understand how my brand is being applied

### 5.2 Logo Color Extraction

```javascript
async function extractColorsFromLogo(logoUrl) {
  // Load image and analyze pixels
  const colors = await analyzeImageColors(logoUrl);

  // Cluster into distinct colors
  const clusters = kMeansClustering(colors, 5);

  // Sort by prominence
  const sorted = clusters.sort((a, b) => b.count - a.count);

  return {
    primary: sorted[0]?.hex,
    secondary: sorted[1]?.hex,
    accent: sorted[2]?.hex,
    // Generate complementary colors if needed
    complementary: generateComplementary(sorted[0]?.hex)
  };
}
```
